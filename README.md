# EverythingTT

This is a simple static website project for "EverythingTT".

## Project Structure

- `index.html`: The main entry point of the website.
- `home-style.css`: Stylesheet for the website.
- `home-scripts.js`: JavaScript logic for the website.
- `.github/workflows/org.yml`: GitHub Actions workflow for deploying to GitHub Pages.

## How to Run Locally

Simply open `index.html` in your web browser.
