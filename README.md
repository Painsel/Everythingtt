# EverythingTT - News Dashboard

A secure, decentralized news dashboard and community platform powered by GitHub API.

## 🚀 Features

- **Secure Backend**: Multi-layered security including IP pinning, session sanitization, and role-based access control.
- **AI Defense**: Integrated Anti-AI protocols (PoW, behavioral biometrics, and prompt injection) to prevent unauthorized scraping and automated attacks.
- **Decentralized Storage**: Utilizes GitHub repositories for encrypted data storage and management.
- **Admin Panel**: Robust tools for article management, account auditing, and security monitoring.

## 🛡️ Security

This project implements advanced security measures:
- **Proof-of-Work (PoW)**: Required for all write operations to block low-cost bot attacks.
- **Behavioral Biometrics**: Detects human-like interaction to filter out automated agents.
- **Dynamic CSP**: Strict Content Security Policy injected at runtime.
- **Encrypted Storage**: Sensitive data is protected using AES/TripleDES encryption.

## 📜 Legal & AI Policy

This project distinguishes between **Authorized Development** and **Malicious Automation**:
- **Permitted**: The use of AI-assisted development tools (e.g., Trae, GitHub Copilot, Gemini) by authorized contributors to build, maintain, and audit the codebase.
- **Prohibited**: The use of automated scripts, scrapers, or adversarial AI agents to bypass security, scrape user data, or conduct denial-of-service attacks against the production environment.

Unauthorized access to private data storage is considered a cyber-security violation.
