Overview

The EverythingTT-v1-preview API allows developers to programmatically communicate with our "Deep Reasoning" engine. It is designed to bridge the gap between low-level game mechanics and high-level market economics.

Every request to the API is processed through the Painsel Engine v6.0, which automatically applies game-specific knowledge including Interest Calculation, Purge Logic, and Wiki-verified economics.

API Endpoints

POST https://kecia-ungreeted-neologically.ngrok-free.dev/v1/chat/completions

The primary entry point for EverythingTT-v1-preview chat completions. Supports OpenAI-compatible message structures.

Request Structure

The API expects a JSON body with the following parameters:

Parameter Type Description
model string Must be set to painsel/EverythingTT-v1-preview.
messages array List of message objects: { "role": "user", "content": "..." }.
temperature float Controls randomness (Recommended: 0.5).
max_tokens int Maximum length of the generated response.

Code Examples

cURL

curl https://kecia-ungreeted-neologically.ngrok-free.dev/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "model": "painsel/EverythingTT-v1-preview",
    "messages": [
      {"role": "user", "content": "Analyze a 0-gold account risk."}
    ],
    "temperature": 0.5
  }'

Python

import requests

endpoint = "https://kecia-ungreeted-neologically.ngrok-free.dev/v1/chat/completions"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}

payload = {
    "model": "painsel/EverythingTT-v1-preview",
    "messages": [
        {"role": "user", "content": "What are the market rates for ETT?"}
    ],
    "temperature": 0.5
}

response = requests.post(endpoint, headers=headers, json=payload)
print(response.json()['choices'][0]['message']['content'])

Node.js

const fetch = require('node-fetch');

const response = await fetch('https://kecia-ungreeted-neologically.ngrok-free.dev/v1/chat/completions', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_TOKEN'
    },
    body: JSON.stringify({
        model: 'painsel/EverythingTT-v1-preview',
        messages: [{ role: 'user', content: 'Explain name prestige bonuses.' }],
        temperature: 0.5
    })
});

const data = await response.json();
console.log(data.choices[0].message.content);
